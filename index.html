夹肉记

发表于 2021-11-16 更新于 2022-03-03 分类于 流萤

　　想把波西尼亚挂毯挂到椭圆机后面的墙上，得先找无痕钉。

　　拉开鞋柜上层的抽屉翻翻，没有。鞋柜台面上放着两个摞起来的彩色的小抽屉盒子，再到这里找找。小盒子放在鞋柜台面上靠墙靠餐桌一侧的角落，鞋柜本身的抽屉还开着没关，顶在那。为了看盒子里的东西，便要俯身向前抻脖往盒子里面瞅。身子往前这么一靠，就把鞋柜抽屉靠关上了。这抽屉关闭时带着挺大劲儿的阻尼，快关上时会使劲往里吸。抽屉外立面和鞋柜立面是一平的，台面也没有沿。在抽屉关闭的最后，开口即将闭合时，抽屉面板上沿内侧和台面下沿之间的缝隙，夹到了肉！因为此时身子还靠在抽屉上！疼得连忙一退，硬是从越夹越紧的缝隙里把皮肉抽了出来。嘶～

　　真是一个可载入糗事百科的操作。

燃气热水器放水记

发表于 2021-11-10 更新于 2022-03-03 分类于 流萤

引子

　　不停地下了两天雪，也算几年不遇的大雪了，积雪平均能有个二十多厘米。

起因

　　燃气热水器厂家发来的短信，提醒说“室外温度零度以下时，要把热水器内部的水放掉，防止冻裂。”因为热水器在室内，有暖气，需不需要放水？搜了一下，在北京也有冻裂的。晚上冷风倒灌，年久管脆。为了以防万一，还是放一下吧。

　　每天用完都得放水，用时再把阀打开。看说明书的步骤似乎也不算太麻烦。

经过

　　晚上洗完澡，开始放水吧。把进水阀拧关上，把电断了。找个盆，把泄水堵头拧开一点，水是滋出来的，再拧点角度，这回是往出冒的了，顺着管子往下淌，再调调……嗵！堵头被冲下来了，水像从水龙头全开一样往外涌。不是说存水靠重力流出来吗？不对劲。先关总水阀，就在脚边水槽底下。Ｚ也发现声音不正常了，跑来帮忙把阀门关了。

　　此时，厨房已经满地满台满墙的水了。赶紧找抹布擦擦擦……忙活半天，真是没事找点事活动活动。

结果

　　左边的才是进水阀，我只关了右边的回水阀。关总阀时意识到了问题所在，手忙脚乱地有点慌，第一时间竟没想着关掉另一个。再看有人帮忙来了，就又挺了一会儿等关总阀了，毕竟绝对有效。

　　为什么不两个阀全关上，这么节约？脑子不够用了。

后记

　　正常放水操作，用水龙头放水大概放个几分钟就完了。泄水堵头是进气的，进气口不大，水放得也慢。进气口咕嘟咕嘟的流不出几滴水，擦一下就行。开阀上水时，各个水龙头的热水会剧烈的喷几下气，要注意一下。
